# Mortgage Overpay

Mortgage Overpay is a mobile-first Next.js website for UK homeowners who want instant answers about mortgage overpayments.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Recharts for lightweight results visualisation

## Local development

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:3000`

## Notes

- Version 1 is static-first and client-side only.
- Mortgage calculations are illustrative and based on a repayment mortgage model.
- Ad placeholders are included for future insertion points without affecting the main calculator journey.
